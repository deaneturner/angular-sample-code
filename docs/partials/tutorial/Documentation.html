Documentation

ngdocs

grunt-ngdocs is a tool used to generate documents from code API annotations and markdown files (.ngdoc).

  • To generate the documentation for this project:

    grunt ngdocs

    or

    grunt dev

    or

    grunt serve

  • To view the documentation web project, open the index.html file the generated docs directory.
