userProfile (userProfile in module common )

Description

A service to load and manage user profile information.

Methods

  • getInfo(addConfig)

    Retrieves the employee's information.

    Parameters

    • addConfig – {Object} –

      A HTTP config object (e.g. {params: {id: "001"}}).

  • init()

    Initializes the employee's information and stores it on the root scope.

Example

The user profile is stored on the $rootScope for easy access.

 {{user.info.companyName}}

or

 $rootScope.user.info.companyName
