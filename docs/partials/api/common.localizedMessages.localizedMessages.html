localizedMessages (localizedMessages in module common )

Description

A service used to retrieve localized strings and messages. Provides support for parameter replacement using an additional parameter set.

Dependencies

  • i18nMessages

Methods

  • get(msgKey, [interpolatePparams])

    Gets the translated value, with string replament parameters (if given).

    Parameters

    • msgKey – {string} –
      • The message key value to be translated.
    • [interpolatePparams] – {Object} –
      • The string replacement parameters to be injected into the message key.

    Returns

    {string} –

    The translated message value.

Example

For a message key equal to:

 first.and.last: 'My first name is {(firstName}} and my last is {{lastName}}.'

In JavaScript:

 localizationMessages.get('first.and.last', {
     firstName: 'foo',
     lastName: 'bar'
 });

Will return:

 My first name is foo and my last is bar.
