i18n (filters in module common )

Description

Localization for expressions using a filter.

Usage

In HTML Template Binding

{{ i18n_expression | i18n:[parameters] }}

In JavaScript

$filter('i18n')(msgKey, [parameters])

Parameters

  • msgKey – {string} –
    • The message key value to be translated.
  • [parameters] – {Object} –
    • The string replacement parameters to be injected into the message key.

Returns

{string} –

The translated message value.

Example

For a message key equal to:

 first.and.last: 'My first name is {firstName} and my last is {lastName}.'

In an expression:

 {{ 'first.and.last' | i18n:{ firstName: 'foo', lastName: 'bar'} }}

Will return:

 My first name is foo and my last is bar.
